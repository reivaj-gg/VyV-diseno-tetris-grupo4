## Actividad
Se solicita realizar los diagramas con el modelo, considerando los requisitos planteados en el documento *"Tetris requisitos"*. Bajo el enfoque **MoSCoW**, el alcance del diseño considera los requisitos que el sistema **debe** tener y, opcionalmente, los que **debería** tener.

Esta entrega contiene, en el directorio [`diagrama/`](./diagrama), el **código del diagrama** y la **imagen ya renderizada** del mismo.

## Cómo renderizar el código
El diagrama está escrito en **C4-PlantUML**. Para regenerar la imagen:
1. Abrir [PlantUML Online](https://www.plantuml.com/plantuml) o usar la extensión PlantUML en VS Code.
2. Pegar el contenido de `diagrama/tetris_c4_container.puml`.
3. Exportar como PNG o SVG.
